# Superstore-Sales-Dashboard
Excel dashboard project - Superstore Monthly Sales Report

## 🔍 Project Overview
An interactive Excel dashboard analyzing Superstore sales data 
with charts, pivot tables, and key business insights.

## 📈 Key Highlights
- *Total Orders:* 9,994
- *Total Sales:* $2,297,200
- *Total Profit:* $286,397
- *Total Quantity:* 37,873

## 📊 Dashboard Features
- Region wise sales (Donut Chart)
- Top 5 customers by sales
- Top 10 subcategory wise sales
- Category wise profit analysis
- Monthly sales trend
- Top 10 products by profit
- Top 10 city wise sales

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs 
- Data Analysis

## 📁 Dataset
Superstore Sales Dataset (4 years of data)
